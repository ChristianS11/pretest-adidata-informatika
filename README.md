# Pretest

## Angular Version

v17.0.0

## NX Version

v17.2.8

## Installation

run `npm install`

## Start the app (Dashboard App & CRUD App)

run `npm start`. Open your browser and navigate to http://localhost:4200/. Happy coding!

## Start crud app

run `npm run crud`. Open your browser and navigate to http://localhost:4201/. Happy coding!
